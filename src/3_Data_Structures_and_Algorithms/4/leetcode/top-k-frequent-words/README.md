## Approaches

| Approach       | Time Complexity | Space Complexity |
| -------------- | --------------- | ---------------- |
| quick select   | O(n \* log(n))  | O(n)             |
| sort           | O(n \* log(n))  | O(n)             |
| priority queue | O(n \* log(n))  | O(n)             |
| bucket sort    | O(n)            | O(n)             |
