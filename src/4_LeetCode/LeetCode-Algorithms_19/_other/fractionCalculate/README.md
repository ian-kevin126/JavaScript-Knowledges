# 46.Fraction Calculate

## Description

* 整数`3`表示成`[3]`
* 整数`-3`表示成`[-3]`
* 分数`7/2`表示成`[3, 1, 2]`
* 分数`-7/2`表示成`[-3, -1, 2]`
* 分数`-1/2`表示成`[0, -1, 2]`

## Example

```javascript
Input: [3], [-3], '+'
Output: [6]

Input: [1, 1, 2], [2], '/'
Output: [0, 3, 4]

Input: [-10], [0, -3, 8], '+'
Output: [-10, -3, 8]
```
