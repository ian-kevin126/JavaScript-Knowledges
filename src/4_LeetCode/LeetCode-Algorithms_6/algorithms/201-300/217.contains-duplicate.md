给定一个整数数组，判断是否存在重复元素。

如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

```cpp
示例 1:

输入: [1,2,3,1]
输出: true
```

---

集合

cpp

```cpp
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_set<int> s;
        for (int i=0; i< nums.size(); i++){
            if (s.count(nums[i]) == 1) return true;
            s.insert(nums[i]);
        }
        return false;
    }
};
```

javascript

```javascript
var containsDuplicate = function(nums) {
    let m = {}
    for (let n of nums){
        if (m[n]) return true
        else m[n] = true
    }
    return false
};
```
