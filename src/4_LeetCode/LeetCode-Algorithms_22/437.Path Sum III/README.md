# Path Sum III

给定一个二叉树，其中每个节点包含一个值。

寻找路径使得节点之和等于给定的数。

这个路径不需要从根节点到叶子节点，但是必须是自顶向下的。

树的节点不会超过1000个，值的范围的在-1,000,000与1,000,000之间。

例如:

```
root = [10,5,-3,3,2,null,11,3,-2,null,1], sum = 8

      10
     /  \
    5   -3
   / \    \
  3   2   11
 / \   \
3  -2   1

返回3，路径如下:

1.  5 -> 3
2.  5 -> 2 -> 1
3. -3 -> 11

```
