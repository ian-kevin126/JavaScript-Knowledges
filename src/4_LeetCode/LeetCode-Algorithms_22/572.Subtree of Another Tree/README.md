# Subtree of Another Tree

给定两个非空二叉树 s 和 t,检查树t与s的子树是否有着相同的结构与节点值。s的子树是指包含一个节点和其所有的后代节点。树可以当做自己的子树。

例如:

树s:

```
     3
    / \
   4   5
  / \
 1   2
```

树t:

```
   4
  / \
 1   2
```

返回true

树s:

```
     3
    / \
   4   5
  / \
 1   2
    /
   0
```


树t:

```
   4
  / \
 1   2
```

返回false
