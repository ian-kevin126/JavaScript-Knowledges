# 617. Merge Two Binary Trees

给定两个二叉树，想象用一棵树覆盖另一棵，两棵树中的部分节点可能会覆盖而另一部分不会覆盖。

你需要将它们合并成一棵新的二叉树。合并规则是如果两个节点覆盖，节点的和作为合并后的节点。否则，非空节点被作为新的二叉树的节点。

例如:

Example 1:
Input:
	Tree 1                     Tree 2
          1                         2
         / \                       / \
        3   2                     1   3
       /                           \   \
      5                             4   7
Output:
Merged tree:
	     3
	    / \
	   4   5
	  / \   \
	 5   4   7

注意: 合并的过程是从两个树的根节点开始。

思路:

我的思路是使用树的先序遍历，然后根据规则生成节点，详情见节点。
